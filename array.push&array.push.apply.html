array.push, array.push.apply和array.push.call<

运行下面的代码，提示框会显示字符串“red|blue|green”

//create the array var values = new Array(); //add the values values.push.apply(values, ["red", "blue", "green"]); alert(values.join("|")); //"red|blue|green"

将push.apply方法换成push方法后，我们发现提示框将显示“”

//create the array var values = new Array(); //add the values values.push(["red", "blue", "green"]); alert(values.join("|")); //"red|blue|green"
